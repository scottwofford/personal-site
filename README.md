# Scott Wofford - Personal Website

A modern, responsive personal website showcasing AI safety research, development projects, and technical expertise.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Interactive Navigation**: Smooth scrolling and dynamic navbar effects
- **Project Showcase**: Featured projects with links and descriptions
- **Skills Section**: Comprehensive overview of technical abilities
- **Contact Integration**: Easy ways to connect and collaborate

## Sections

1. **Hero**: Introduction and call-to-action
2. **About**: Focus areas and expertise overview
3. **Projects**: Featured work including Luthien Control Framework
4. **Skills**: Technical skills organized by category
5. **Contact**: Professional contact information and social links

## Technologies Used

- **HTML5**: Semantic markup and accessibility
- **CSS3**: Modern styling with custom properties and animations
- **Vanilla JavaScript**: Interactive functionality and smooth scrolling
- **Font Awesome**: Professional iconography
- **Responsive Design**: Mobile-first approach with CSS Grid and Flexbox

## Local Development

To run the website locally:

```bash
# Navigate to the project directory
cd personal-site

# Start a simple HTTP server
python3 -m http.server 8080

# Open http://localhost:8080 in your browser
```

## Deployment

This is a static website that can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

## Customization

### Colors
The website uses CSS custom properties for easy theming:
```css
:root {
  --primary: #1a1a2e;
  --secondary: #16213e;
  --accent: #e94560;
}
```

### Content
Update the content in `index.html` to reflect your:
- Personal information
- Project details
- Contact information
- Social media links
