Current State (Jan 2026)

We're trying to figure out how to build a thing that makes it possible to put AI control into production. Specifically: building arbitrary policies for an LLM proxy.

Right now we're testing and debugging as things come up.

Status: Neither Scott nor Jai is dogfooding aggressively. Product is too buggy for daily use.

Recent example: Demo to Seldon folks (Jan 24) hit 500 errors from thinking blocks streaming bug. PR #134 fixed it, but the pattern persists. We need demo prep checklists, not just code fixes.

Open question from Jai (Jan 20): Models are getting noticeably better. His pdoom has gone down. Does model improvement reduce the leverage of control work?
